# 学术副词 Hedging & Boosting

> Band 7 要求 "awareness of style"。学术写作需要谨慎措辞（hedging），避免绝对化。
> 同时也要会用 boosting 表达确信立场。

## Hedging 委婉 / 谨慎

arguably / presumably / seemingly / apparently / ostensibly / supposedly / reportedly / allegedly / to some extent / to a certain degree / by and large / in general / for the most part / to my knowledge

**句型**：
- it could be argued that
- it seems likely that
- there is a tendency for
- one might say that

## Boosting 强调 / 确信

undoubtedly / unquestionably / undeniably / clearly / obviously / evidently / certainly / surely / indeed / in fact / as a matter of fact

**句型**：
- it is undeniable that
- there is no denying that
- without a doubt

## 频率

invariably / consistently / habitually / customarily / routinely / occasionally / sporadically / intermittently / seldom / rarely

## 程度

considerably / substantially / significantly / markedly / notably / dramatically / drastically / slightly / marginally / modestly
