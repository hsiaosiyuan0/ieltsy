# 连接词与话语标记 Cohesive Devices

> Coherence and Cohesion 评分的核心给分点。**避免单一重复使用 "Firstly / Secondly / Finally" 这种低分套路。**

## 列举 Listing
- **First**：to begin with / first and foremost / for one thing / initially / the primary reason is that
- **Next**：subsequently / following this / what's more / on top of this / coupled with this
- **Last**：finally / lastly / ultimately / last but not least

## 递进 Addition
moreover / furthermore / in addition / additionally / besides / what is more / on top of that / not to mention / coupled with / along with

## 转折 Contrast
however / nevertheless / nonetheless / on the contrary / on the other hand / conversely / by contrast / having said that / that being said / albeit / whereas / while

## 让步 Concession
admittedly / granted / it is true that / despite the fact that / although / even though / notwithstanding / regardless of

## 因果 Cause & Effect

**因为**：because / since / as / owing to / due to / on account of / in light of / given that / in view of

**所以**：therefore / thus / hence / consequently / as a result / accordingly / for this reason

**导致**：lead to / give rise to / contribute to / result in / bring about / trigger / pave the way for

## 举例 Exemplification
for example / for instance / to illustrate / a case in point is / take ... as an example / namely / specifically / such as

## 强调 Emphasis
indeed / in fact / as a matter of fact / undoubtedly / clearly / obviously / it is worth noting that / what is more striking is

## 比较 Comparison
similarly / likewise / in the same vein / in much the same way / just as / by the same token

## 总结 Conclusion
in conclusion / to conclude / to sum up / in summary / overall / on the whole / all things considered / taking everything into consideration

## 重述 Restatement
in other words / that is to say / put differently / namely / to put it another way

## 条件 Condition
provided that / as long as / on the condition that / unless / in the event that / suppose / supposing

## 时间 Time
meanwhile / simultaneously / subsequently / previously / formerly / hitherto / nowadays / in the past / for the time being
