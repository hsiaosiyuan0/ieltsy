# 高频搭配 Collocations

> Band 7 评分语描述："uses less common and idiomatic vocabulary."
> 自然搭配比单个高级词更能拿分。

## 动词 + 名词

- **make**: make a decision / make an effort / make progress / make a difference / make a contribution / make sense / make sure / make assumptions
- **take**: take action / take responsibility / take advantage of / take into account / take into consideration / take place / take measures
- **have**: have an impact on / have an effect on / have access to / have an influence on / have a tendency to / have the opportunity to
- **do**: do research / do business / do harm / do damage to / do justice to
- **pay**: pay attention to / pay tribute to / pay the price for / pay a visit to
- **draw**: draw attention to / draw a conclusion / draw a comparison / draw a distinction / draw on (knowledge / experience)
- **raise**: raise awareness / raise concerns / raise doubts / raise questions / raise standards / raise the bar
- **play**: play a (vital / crucial / pivotal) role in / play a part in
- **pose**: pose a threat / pose a risk / pose a challenge / pose a problem
- **gain**: gain experience / gain insight into / gain access to / gain popularity / gain momentum
- **shed**: shed light on
- **bear**: bear in mind / bear responsibility / bear fruit
- **strike**: strike a balance / strike a deal / strike a chord with
- **meet**: meet the demand / meet the needs / meet the requirements / meet the deadline / meet expectations
- **set**: set a goal / set an example / set a precedent / set the standard / set in motion

## 形容词 + 名词

- a viable solution / a feasible approach / a sustainable model
- a heated debate / a vibrant community / a vested interest
- ample evidence / compelling evidence / overwhelming evidence
- a fundamental shift / a paradigm shift / a seismic change
- profound impact / far-reaching consequences / lasting effects
- pressing issue / burning question / contentious topic
- adverse effects / detrimental consequences / dire consequences
- a significant proportion / a substantial number / a marginal increase
- unprecedented growth / staggering numbers / soaring prices

## 副词 + 形容词

- highly controversial / highly recommended / highly likely
- widely accepted / widely regarded as / widely believed
- deeply concerned / deeply rooted / deeply ingrained
- closely related / closely linked / closely connected
- strongly opposed / strongly support / strongly suggest
- arguably the most ... / undoubtedly the best ... / unquestionably effective

## 名词 + 介词（搭配难点）

- a solution **to** a problem
- an approach **to** sth
- an increase / decrease **in** sth
- the cause **of** sth / the reason **for** sth
- the impact / effect **on** sth
- access **to** sth
- exposure **to** sth
- a remedy / cure **for** sth
- emphasis **on** sth
- demand **for** sth

## 动词 + 介词（搭配难点）

- account **for** / cope **with** / cater **to** / resort **to**
- contribute **to** / lead **to** / object **to** / subscribe **to**
- comply **with** / interfere **with** / dispense **with**
- benefit **from** / suffer **from** / refrain **from** / stem **from**
- focus **on** / depend **on** / impose ... **on** / impinge **on**
- consist **of** / dispose **of** / approve **of** / boast **of**
