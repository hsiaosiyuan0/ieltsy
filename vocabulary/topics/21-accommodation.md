# Topic 21 — Accommodation & Housing 住房

## 核心名词
- **类型**：accommodation / property / housing / apartment / flat / studio / bungalow / cottage / mansion / townhouse / detached house / semi-detached / terraced house / dormitory / hostel
- **租赁**：lease / mortgage / tenant / landlord / rent / deposit / utilities / amenities / appliance / furnishing / decor / floor plan / square metre
- **房间**：living room / bedroom / kitchen / bathroom / hallway / corridor / staircase / attic / basement / cellar / balcony / patio / garage / garden / yard
- **家具/电器**：sofa / couch / armchair / wardrobe / dresser / nightstand / cabinet / shelf / mattress / appliance / oven / microwave / fridge / dishwasher

## 核心动词
rent / lease / sublet / move in / move out / settle in / share / co-habit / refurbish / renovate / decorate / furnish / declutter / tidy up

## 核心形容词
- **空间**：spacious / roomy / cosy / cramped / cluttered / open-plan
- **采光通风**：well-lit / dim / airy / stuffy / sunlit
- **位置**：centrally located / well-connected / secluded / remote
- **状态**：well-maintained / run-down / dilapidated / newly-built / freshly-painted
- **配置**：furnished / unfurnished / fully-equipped / self-contained

## 高分搭配 / 习语
- feel at home / make oneself at home
- a roof over one's head
- split the rent / share the bills
- put down roots（安家）
- there's no place like home
- a stone's throw from（离...很近）
