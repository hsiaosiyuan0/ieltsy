# Topic 17 — Landscape & Scenery 风景与自然地貌

## 核心名词
- **山地**：landscape / scenery / terrain / mountain / hill / valley / cliff / peak / summit / plateau / ridge / slope / canyon / gorge
- **平地**：plain / meadow / pasture / prairie / steppe
- **干旱**：desert / dune / oasis
- **林地**：forest / woodland / jungle / rainforest / grove
- **水域**：river / stream / brook / waterfall / lake / pond / pool / spring / wetland / swamp / marsh
- **海岸**：coast / shore / beach / bay / harbour / cape / peninsula / island / archipelago / reef
- **特殊**：cave / cavern / glacier / iceberg / volcano / crater

## 核心动词
stretch (out) / overlook / dominate / surround / rise / tower (over) / wind / meander / flow / cascade / span

## 核心形容词
breathtaking / picturesque / scenic / spectacular / stunning / majestic / awe-inspiring / serene / tranquil / lush / verdant / barren / rugged / rocky / vast / sprawling / rolling / steep / pristine / unspoilt

## 高分搭配 / 习语
- a panoramic view / a bird's-eye view
- take one's breath away
- a feast for the eyes
- as far as the eye can see
- nestled in the mountains
- off the beaten track
