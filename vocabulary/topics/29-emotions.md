# Topic 29 — Emotions & Feelings 情感

## 核心名词
emotion / feeling / mood / sentiment / sensation / impulse / temper / spirits

## 正面情绪 Positive

### 名词
joy / delight / contentment / gratitude / relief / pride / affection / enthusiasm / satisfaction / euphoria / serenity

### 形容词
- **强烈喜悦**：thrilled / overjoyed / ecstatic / elated / euphoric
- **平和满足**：content / pleased / satisfied / serene / at peace
- **感激**：grateful / thankful / indebted
- **兴奋**：enthusiastic / passionate / eager / pumped

## 负面情绪 Negative

### 名词
anger / fury / rage / frustration / annoyance / irritation / sadness / grief / sorrow / despair / anxiety / dread / fear / envy / jealousy / regret / guilt / shame / embarrassment / humiliation

### 形容词
- **愤怒**：furious / livid / outraged / annoyed / irritated / frustrated / fed up
- **悲伤**：devastated / heartbroken / miserable / gloomy / down / depressed
- **焦虑/害怕**：anxious / nervous / uneasy / apprehensive / terrified / petrified / panicked
- **不满**：disappointed / disillusioned / discouraged
- **羞愧**：embarrassed / ashamed / humiliated / mortified
- **复杂**：overwhelmed / conflicted / torn / numb / indifferent / envious / resentful

## 核心动词
experience / feel / express / suppress / vent / overcome / cope with / get over / lift one's spirits / boost morale / hold a grudge / bottle up / let out

## 高分搭配 / 习语
- be on cloud nine（极度开心）
- over the moon（开心翻天）
- on top of the world
- down in the dumps（情绪低落）
- in low spirits
- butterflies in one's stomach（紧张）
- a lump in one's throat（哽咽）
- lose one's temper / fly off the handle
- keep one's cool / hold it together
- mixed feelings（百感交集）
