# Topic 25 — Sports & Fitness 运动健身

- **水上**：swimming / diving / surfing / sailing / rowing / kayaking
- **耐力**：jogging / running / marathon / cycling / triathlon
- **力量/训练**：weightlifting / yoga / pilates / aerobics / CrossFit / HIIT
- **户外**：hiking / climbing / skiing / snowboarding / skating
- **格斗**：martial arts / boxing / judo / karate / taekwondo

## 核心形容词

- a level playing field（公平竞争）
- give it your all
- push oneself to the limit

- throw in the towel（认输）
