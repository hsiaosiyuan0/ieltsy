# Topic 16 — Weather 天气

## 核心名词
weather / climate / forecast / temperature / humidity / breeze / gale / drizzle / downpour / shower / thunderstorm / lightning / hailstorm / blizzard / fog / mist / haze / drought / heatwave / cold snap / frost / dew

## 核心动词
pour (down) / drizzle / clear up / brighten up / freeze (over) / thaw / heat up / cool down / settle in / die down

## 核心形容词
- 热：scorching / boiling / sweltering / sizzling / sultry
- 冷：freezing / chilly / nippy / bitter / icy
- 舒适：mild / pleasant / crisp / balmy
- 湿度：humid / muggy / damp / dry / arid
- 天空：sunny / overcast / cloudy / clear / hazy
- 风：breezy / windy / blustery / gusty / still / calm
- 其他：stormy / foggy / misty / drizzly

## 高分搭配 / 习语
- under the weather（不舒服）
- weather the storm（挺过难关）
- a glorious day / a miserable day
- take a turn for the worse
- be caught in a downpour
- bundle up against the cold
