# Topic 24 — Hobbies & Leisure 兴趣爱好

## 核心名词
hobby / pastime / leisure / pursuit / interest / passion / craft / collection / enthusiast / amateur

### 常见爱好
- **手工类**：gardening / knitting / sewing / pottery / calligraphy / origami / scrapbooking / DIY (do-it-yourself) / woodworking
- **艺术类**：photography / painting / drawing / sketching / sculpting / playing an instrument
- **数字类**：coding / blogging / vlogging / video editing / gaming / podcasting
- **户外类**：hiking / camping / fishing / birdwatching / stargazing / cycling
- **室内类**：reading / collecting / cooking / baking / board games / puzzles / chess / volunteering

## 核心动词
pursue / take up / give up / pick up / dabble in / be into / be hooked on / be obsessed with / unwind / relax / recharge / kill time / pass the time / engage in / immerse oneself in

## 核心形容词
rewarding / fulfilling / engaging / absorbing / time-consuming / addictive / therapeutic / relaxing / stimulating / mind-numbing

## 高分搭配 / 习语
- hone one's skills（磨练技能）
- kill time（打发时间）
- a means of escape（逃离日常）
- recharge one's batteries（充电）
- strike a balance between work and leisure
- get lost in（沉浸其中）
- a hobby that runs in the family
- right up my alley（正合我意）
