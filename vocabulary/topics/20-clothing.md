# Topic 20 — Clothing & Fashion 服装时尚

## 核心名词
outfit / attire / garment / wardrobe / accessory / fabric / pattern / trend / brand / designer label / fashion / collection / runway

### 服装单品
- **正式**：suit / blazer / waistcoat / vest / tie / bow tie / blouse / dress / gown / formal wear
- **休闲**：T-shirt / hoodie / sweatshirt / cardigan / jeans / shorts / skirt / leggings / tracksuit
- **外套**：coat / jacket / parka / overcoat / windbreaker / raincoat
- **鞋类**：sneakers / trainers / boots / heels / flats / loafers / sandals / flip-flops
- **配饰**：scarf / hat / cap / belt / glove / sunglasses / handbag / wallet / watch / jewellery / necklace / bracelet / ring / earring

### 面料 Materials
cotton / linen / silk / wool / cashmere / denim / leather / suede / velvet / lace / synthetic / polyester / nylon

### 图案 Patterns
plain / striped / checked / plaid / floral / polka-dot / paisley

## 核心动词
wear / dress (up / down) / try on / put on / take off / slip into / suit / fit / match / go with / clash with / accessorize / iron / fold / hang up

## 核心形容词
- **风格**：fashionable / trendy / stylish / chic / elegant / sophisticated / smart / casual / sporty / sloppy / scruffy / outdated / timeless / vintage
- **版型**：form-fitting / loose-fitting / baggy / oversized / tailored / snug / tight
- **颜色**：bold / muted / vibrant / pastel / neutral / earthy

## 高分搭配 / 习语
- dress to impress
- dress code（着装要求）
- make a fashion statement
- a wardrobe staple（衣柜必备）
- be in vogue（流行）/ be out of fashion
- fit like a glove
- on a shoestring（节俭）
