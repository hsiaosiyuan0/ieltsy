# Topic 27 — Hometown 家乡

## 核心名词
hometown / birthplace / native place / region / district / suburb / outskirts / city center / downtown / neighbourhood / community / population / inhabitant / resident / local / heritage / charm / landmark / monument

### 城市设施
infrastructure / amenities / public facilities / park / square / plaza / library / community centre / hospital / school district / market

## 核心动词
be born and raised in / grow up in / settle / reside / migrate / relocate / be rooted in / leave / return to

## 核心形容词
- **氛围**：bustling / vibrant / lively / dynamic / quiet / sleepy / laid-back / peaceful
- **风貌**：charming / picturesque / quaint / historic / industrial / commercial / residential
- **文化**：multicultural / cosmopolitan / diverse / traditional / conservative
- **发展**：developed / underdeveloped / up-and-coming / thriving / declining

## 高分搭配 / 习语
- a sense of belonging（归属感）
- put down roots（扎根）
- a tight-knit community（紧密的社区）
- a melting pot of cultures
- off the beaten track
- there's no place like home
- be steeped in history（充满历史底蕴）
- hidden gem（隐藏的宝藏）
