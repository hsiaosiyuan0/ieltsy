## §21 短语动词 Phrasal Verbs

> Speaking Band 7 要求 "idiomatic vocabulary"，phrasal verbs 是核心。

### 类型分类
231. Type 1 — 不及物 ★★★（break down / take off / get up / show up）
232. Type 2 — 及物可分 ★★★（turn on / put off / pick up；代词必须放中间：turn it on ✓ / turn on it ✗）
233. Type 3 — 及物不可分 ★★★（look after / count on / deal with）
234. Type 4 — 三词短语动词 ★★ — `look forward to / put up with / get along with / come up with` 整体不可拆；末尾 `to` 若为介词，后接名词或 `V-ing`：`look forward to seeing you`

### IELTS 高频短语动词（按主题）
235. 学术 / 思维类 ★★★ — account for / bring about / point out / set out / refer to / draw on / sum up / look into / come up with / figure out / work out / call for
236. 社会 / 交际类 ★★ — get along with / look up to / look down on / bring up / put up with / fall out with / fit in
237. 变化 / 发展类 ★★★ — bring about / give rise to / lead to / result in / set off / set in / kick off / take off
238. 应对 / 处理类 ★★★ — deal with / cope with / put up with / get over / get through / sort out / take on / carry out / handle
239. 成长 / 学习类 ★★ — grow up / take after / take up / pick up / brush up on / catch up with / keep up with
240. 工作 / 行动类 ★★★ — carry out / set up / take over / give up / put off / call off / turn down / work out / run out of
241. 时间 / 状态类 ★★ — come up / come about / come across / come down to / end up / turn out / break out

### 短语动词 vs 单动词的语体差异
242. 短语动词偏口语，拉丁词源单动词偏书面 ★★ — put off ↔ postpone / find out ↔ discover / look into ↔ investigate
243. 学术写作中正式场合优先单动词；Speaking 中适度用短语动词显地道 ★★

## §22 动词补足模式 Verb Patterns

> 哪个动词后接 to do、哪个接 V-ing、哪个接 that-clause —— 完全靠记忆，但这是 Band 7 高频丢分点。

### 名词性补足
244. V + that-clause ★★★ — think / believe / say / claim / argue / suggest / hope / realize（that 可省）
245. V + wh-clause ★★ — wonder / ask / know / realize / decide / discuss
246. 强调 / 建议类 + that S (should) V ★★ — `suggest / propose / recommend + V-ing` 或 `that S (should) V`；`advise / urge + O + to V`，不能写 `suggest sb to do`

### 不定式与动名词
247. V + to-inf ★★★ — `want / hope / decide / promise / agree / refuse / manage / offer / aim / intend + to V`；常用于计划、意愿、承诺与一次行动
248. V + V-ing ★★★ — `enjoy / mind / avoid / suggest / consider / deny / admit / postpone / risk / finish + V-ing`；把动作当作活动、经验或事实讨论
249. V + to-inf 或 V + V-ing 意义不同 ★★★ — remember / forget / regret / stop / try / mean / go on / propose
250. V + O + to-inf ★★★ — `ask / tell / expect / advise / encourage / persuade / allow / warn / invite / remind + O + to V`；宾语必须明确，`say me to` 和 `suggest me to` 都不成立
251. V + O + bare-inf ★★ — make / let / have（使役）+ 感官动词 see / hear / watch / feel / notice / observe + V（动作完整）
252. V + O + V-ing ★★ — see / hear / watch / catch / find / keep + sb V-ing（动作正在进行）
253. V + O + V-ed ★★ — have / get sth done（被动使役）/ see / hear sth done

### 复合补足
254. V + O + adj/n（认为类）★★★ — find sth difficult / consider sb a friend / call sb stupid / make sb happy / believe sth true / declare sth illegal
255. V + O + O 双宾语 ★★ — give / send / show / lend / teach / tell / offer / hand / pass
256. V + O + prep + sth ★★★ — blame sb for / accuse sb of / remind sb of / inform sb of / convince sb of / congratulate sb on / thank sb for / provide sb with / supply sb with / rob sb of / deprive sb of

## §23 使役结构 Causatives

257. `have sth done` ★★★ — 让某事被做（中性）：I had my hair cut.
258. `get sth done` ★★ — 同上（口语化）：I got my watch fixed.
259. `have sb do` ★★ — 直接安排某人做：I had the mechanic check the brakes.
260. `get sb to do` ★★ — 说服某人做：I got my brother to help me.
261. `make sb do` ★★★ — 强迫：The teacher made us rewrite the essay.
262. `let sb do` ★★★ — 允许：My parents let me stay out late.
263. `help sb (to) do` ★★ — 帮助（to 可省）
264. 使役结构的被动 ★★ — `be made to do`（强迫被动有 to）/ `be allowed to do` / `be seen to do`

## 语法笔记 Grammar Notes

### 234. Type 4 — 三词短语动词

#### 核心判断

三词短语动词通常由“动词 + 副词/介词 + 介词”构成，三个部分共同决定意义，宾语放在整个结构之后。末尾出现 `to` 时，先确认它是不是介词。

- `look forward to + noun / V-ing`：期待。
- `put up with + noun / V-ing`：忍受。
- `get along with + person`：与某人相处。
- `come up with + idea / solution`：想出。

#### 生活语境

- 家庭：`I can't put up with leaving dirty dishes overnight.`
- 工作：`We need to come up with a simpler schedule.`
- 社交：`My children get along with our new neighbours.`
- 出行：`I'm looking forward to visiting Hangzhou next month.`

#### 最小对立与纠错

- `I want to meet you.`：`to` 是不定式标记，接原形 `meet`。
- `I look forward to meeting you.`：`to` 是介词，接 `meeting`。
- `I look forward to meet you.` ✗ → `I look forward to meeting you.` ✓
- `We came a solution up with.` ✗ → `We came up with a solution.` ✓

#### 语域迁移

这些结构在自然口语中很常见；正式写作可以按语境换成 `tolerate / devise / anticipate`，但替换前要确认意义和搭配没有改变。

### 246. 强调 / 建议类 + that S (should) V

#### 核心判断

转述建议时，中心动词决定后面能否出现宾语、动名词、不定式或 `that` 从句。不要把所有中文“建议某人做”都套成 `verb + sb + to do`。

- `suggest / recommend doing`：提出某项活动。
- `suggest / recommend that S (should) V`：提出完整建议，美式英语常用原形虚拟式。
- `advise / urge / encourage sb to V`：直接指出建议或推动的对象。
- `promise / offer / refuse to V`：转述说话人自己的承诺、提议或拒绝。
- `admit / deny V-ing`：承认或否认已经发生或被指控的行为。

#### 生活语境

- 看病：`The doctor advised me to rest and suggested working from home.`
- 工作：`My manager recommended that we start the meeting earlier.`
- 家庭：`I promised to collect the parcel after work.`
- 购物：`The shop offered to replace the kettle but denied causing the damage.`

#### 最小对立与纠错

- `She suggested taking the bus.` ✓
- `She suggested that we take the bus.` ✓
- `She suggested us to take the bus.` ✗ → `She advised us to take the bus.` ✓
- `He said me to wait.` ✗ → `He told me to wait.` ✓ / `He said that I should wait.` ✓

#### 语域迁移

口语里 `suggest doing` 简洁自然；正式邮件和论证写作常用 `recommend that...`。选择结构的首要依据是转述行为和参与者关系，不是句子长短。
