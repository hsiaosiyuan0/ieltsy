# 第 5 章：倒装、强调与衔接 Inversion · Emphasis · Cohesion

## §8 倒装 Inversion

102. 否定词置于句首 ★★ — `Never` / `Hardly` / `Seldom` / `Rarely` / `Not only` / `Little` / `No sooner` / `Nor` / `Neither` / `Under no circumstances` / `In no way`
103. `only` + 状语置于句首 ★★
104. `so` / `neither` / `nor` 的承接倒装 ★★
105. 条件句倒装 ★★ — `Had I known...` / `Were I to...` / `Should you...`
106. `so / such ... that` 倒装 ★ — `So great was his anger that...`
107. 表语前置倒装 ★ — `Gone are the days when...`
108. 地点 / 方位状语前置 ★ — `In the corner stood a chair.`
109. 半倒装（助动词提前）vs 全倒装 ★★

## §9 强调与分裂 Emphasis & Cleft

110. `It is/was ... that/who...` 强调句 ★★★
111. `What ... is/was ...` 分裂句 ★★
112. `All (that) ... is/was ...` ★
113. `do / does / did` 强调谓语 ★★
114. `very` / `the very` + 名词 ★
115. `indeed` / `at all` / `whatsoever` 等强调副词 ★

## §10.1 并列连词 Coordinating

116. FANBOYS ★★★ — for / and / nor / but / or / yet / so（前面加逗号的规则）

## §10.2 关联连词 Correlative

117. `both A and B` / `either A or B` / `neither A nor B` / `not only A but also B` / `not A but B` / `whether A or B` ★★★
118. 关联连词的平行结构 ★★

## §10.3 副词性连接 Conjunctive Adverbs

119. 转折 ★★★ — however / nevertheless / nonetheless / on the contrary / on the other hand / conversely
120. 因果 ★★★ — therefore / thus / hence / consequently / as a result / accordingly
121. 递进 ★★★ — moreover / furthermore / in addition / besides / what's more
122. 列举 ★★ — first / firstly / for one thing / for another / finally / lastly
123. 举例 ★★★ — for example / for instance / namely / such as / specifically / to illustrate
124. 总结 ★★ — in conclusion / to sum up / in short / overall / on the whole
125. 让步 ★★ — admittedly / granted / it is true that

## §10.4 介词性连接

126. `despite` / `in spite of` + n. ★★★（vs `although` + 句子）
127. `due to` / `owing to` / `because of` / `as a result of` / `thanks to` ★★★
128. `in addition to` / `apart from` / `besides` + n. ★★

## §10.5 指代与替代衔接

129. 指代 ★★★ — this / that / these / those / it / one(s) / such
130. 替代 ★★ — `so` / `not` / `do so` / `do that`
131. 省略 ★

## §33 口语 Discourse Markers

> 不用于书面写作。Speaking 中适度使用提升 Fluency 评分，但不要塞满。

339. 思考填充 ★★ — `well` / `um` / `er` / `let me think` / `how should I put it` / `you see`
340. 重述 / 解释 ★★ — `I mean` / `what I mean is` / `what I'm trying to say is`
341. 软化语气 ★ — `sort of` / `kind of` / `a bit` / `I guess` / `I suppose` / `probably` / `maybe`
342. 立场标识 ★ — `to be honest` / `honestly speaking` / `frankly`
343. 事实强调 ★ — `actually` / `in fact` / `as a matter of fact`
344. 评价标记 ★ — `surprisingly` / `interestingly` / `unfortunately`
345. 话题转换 ★ — `by the way` / `speaking of which` / `that reminds me` / `anyway` / `anyhow`
346. 听话人回应 ★ — `you know` / `you see` / `right` / `fair enough` / `makes sense`
