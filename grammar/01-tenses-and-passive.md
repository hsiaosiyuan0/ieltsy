# 第 1 章：时态与被动 Tenses & Passive

> 共 29 点｜★★★ 13 / ★★ 11 / ★ 5

## §1 时态 Tenses

### 一般时 Simple
1. 一般现在时 ★★★ — 真理 / 习惯 / 状态
2. 一般过去时 ★★★ — 过去动作 / 状态
3. 一般将来时 ★★★ — `will` / `be going to` / `shall`
4. 过去将来时 ★★ — `would` / `was going to`

### 进行时 Continuous
5. 现在进行时 ★★★ — 当下 / 临时 / 抱怨（always doing）
6. 过去进行时 ★★★ — 过去某点正在 / 背景动作
7. 将来进行时 ★★ — `will be doing`（自然发生 / 礼貌询问）
8. 过去将来进行时 ★

### 完成时 Perfect
9. 现在完成时 ★★★ — 与现在相关 / `since` / `for` / `already` / `yet` / `ever` / `just`
10. 过去完成时 ★★★ — 过去的过去 / `by the time` / `before`
11. 将来完成时 ★★ — `will have done by ...`
12. 过去将来完成时 ★

### 完成进行时 Perfect Continuous
13. 现在完成进行时 ★★★ — 持续到现在 / 强调过程（**Band 7 标志性时态**）
14. 过去完成进行时 ★★ — 持续到过去某点
15. 将来完成进行时 ★

### 时态选择难点
16. `used to do` vs `be used to doing` vs `would do` ★★★
17. 现在完成 vs 一般过去 ★★★（British vs American 倾向）
18. 主从句时态呼应（sequence of tenses） ★★
19. 时间状语从句中"主将从现" ★★★

## §2 被动语态 Passive Voice

20. 一般时被动 ★★★
21. 进行时被动 ★★ — `is being done`
22. 完成时被动 ★★ — `has been done`
23. 情态动词被动 ★★ — `must be done` / `should have been done`
24. 不定式被动 ★★ — `to be done` / `to have been done`
25. 动名词被动 ★ — `being done` / `having been done`
26. 双宾语被动 ★★（She was given a prize / A prize was given to her）
27. `get` + 过去分词 ★（informal passive）
28. 学术写作中"被动 + by sb" 与 "主动" 的选择 ★★
29. `It is said / believed / reported that...` 客观表达句型 ★★★
