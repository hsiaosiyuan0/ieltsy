## §31 否定模式 Negation

### 部分否定 vs 完全否定
325. 部分否定 ★★ — `not all / not every / not always / not both / not entirely / not necessarily`（"不都...""不总是..."）
326. 完全否定 ★★ — `none / no / nobody / nothing / never / neither`

### 否定前缀 / 后缀
327. 否定前缀 ★★ — un- / in- / im- / il- / ir- / non- / mis- / dis-
328. 否定后缀 ★ — -less

### 否定的特殊形式
329. 形式上双重否定但语义肯定 ★ — `It is not uncommon.`（= 相当常见）/ `I cannot but agree.`（= 不得不同意）
330. 否定转移 ★★ — `I don't think he's right.`（思想动词后否定前移）适用 think / believe / suppose / imagine / expect
331. 半否定词 ★★ — `hardly / scarcely / seldom / rarely / barely / few / little`（出现时不再加 not，置于句首引发倒装）

## §32 省略与替代 Ellipsis & Substitution

332. 助动词省略 ★ — `I can swim, but he can't (swim).`
333. to-不定式省略 ★ — `I'd like to go, but I don't want to (go).`
334. 主语省略 ★ — `Hope to see you soon.`（口语 / 非正式）
335. 重复成分省略 ★ — `He likes coffee, and (he likes) tea.`
336. 名词替代 one / ones ★★ — `Do you want a red one?`
337. 句子替代 so / not ★★ — `I think so.` / `I hope not.` / `I'm afraid so.`
338. do so / do that 替代动词短语 ★ — `He told me to leave, so I did so.`

## §34 固定句型 Fixed Patterns

### There be 类
347. `there is/are` ★★★
348. `there seems to be / appears to be / happens to be` ★★ — 推测
349. `there used to be` ★ — 过去存在
350. `there must / might / can be` ★ — 情态

### It 句型
351. `It is + adj + (for sb) + to do` ★★★
352. `It is + adj + that-clause` ★★★
353. `It takes (sb) + 时间 + to do` ★★★
354. `It is no use / no good + V-ing` ★★ — 没用做
355. `It's worth + V-ing` ★★ — 值得（worthwhile 后接 to do 或 V-ing）
356. `It is about / high time + S + V (过去式)` ★ — 早该
357. `It seems / appears / turns out + that` ★★

### Have + 名词类（口语化"做某动作一次"）
358. `have a look / try / rest / swim / chat` ★★ — 比直接动词更口语自然

### There's no 类
359. `There's no point (in) V-ing` ★★ — 没必要
360. `There's no need to V` ★★ — 没必要
361. `There's no harm in V-ing` ★ — 无害
362. `There's no doubt that` ★★ — 毫无疑问

### Cannot help / Cannot but 类
363. `can't help V-ing` ★★ — 情不自禁
364. `can't help but V` ★ — 不得不
365. `cannot wait to V` ★ — 迫不及待

### Would rather / Had better 类
366. `would rather V than V` ★★ — 宁愿
367. `would rather S + V (过去式)` ★ — 希望某人...
368. `had better (not) V` ★★ — 最好
369. `may / might (just) as well V` ★ — 不妨

### So / Such 类
370. `so + adj/adv + that` / `such (a/an) + (adj) + n + that` ★★★
371. `so + adj + a + n + that` ★ — `so beautiful a girl that...`（高级用法）
372. `so as to do` / `so as not to do` ★★ — 目的

### The + 比较级类
373. `The + 比较级, the + 比较级` ★★ — 越...越...
374. `as + 形容词 + as possible` ★★ — 尽可能
375. `not so much A as B` ★ — 与其说A不如说B
376. `A is to B what C is to D` ★ — A对B正如C对D（高级写作）
