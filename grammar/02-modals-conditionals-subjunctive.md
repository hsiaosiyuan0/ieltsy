## §3 情态动词 Modal Verbs

### 推测 / 可能性 (Epistemic)
30. `must` / `can't` ★★★ — 强烈推测
31. `may` / `might` / `could` ★★★ — 可能（**hedging 核心**）
32. `should` / `ought to` ★★ — 推测应该
33. 完成式情态 ★★ — `must have done` / `could have done` / `might have done` / `should have done` / `needn't have done`

### 义务 / 允许 (Deontic)

35. `should` / `ought to` / `had better` ★★
36. `can` / `could` / `may` ★★★ — 允许
37. `need` / `needn't` / `don't need to` ★★

## §40 能力、互动与确定性 Ability · Interaction · Certainty

401. 一般与未来能力：`can / could / be able to` ★★★ — `can / could` 表一般能力；其他时体及不定式环境用 `be able to`；例：`I can swim now, and I will be able to dive after the course.`
402. 过去能力：`could / was able to / managed to` ★★★ — `could` 常表过去一般能力，肯定句中的一次成功多用 `was able to / managed to`；例：`She could read early, and yesterday she managed to finish the test.`
403. `will / would / could` 的意愿、请求与提议 ★★★ — 用 `Will you…? / Would you mind…? / Could I…? / I would like…` 按关系与语域调节礼貌度；例：`Would you mind sending me the details?`
404. 确定性与概率梯度 ★★★ — `definitely / probably / may well / be likely or unlikely to / be bound to` 对现在和未来预测分级；例：`Demand will probably rise, but it is unlikely to double.`

## §41 半情态与词汇情态 Semi-modals & Lexical Modality

447. 义务、预期与安排：`be supposed / expected / meant / set to` ★★★ — 区分规范、外部预期、原定目的和已经安排的未来；例：`The new rule is supposed to protect tenants and is set to take effect in June.`
448. `need` 与 `dare` 的情态 / 实义动词用法 ★★ — 情态用法直接接原形且常见于否定、疑问，实义动词按普通动词变化；例：`You need not reply today, and no one dared to interrupt.`
459. 词汇化情态：名词、形容词与副词 ★★ — 用 `possibility / likelihood / likely / possible / perhaps / probably / tend to` 在不同语体中表达判断强度；例：`There is a strong possibility that demand will decline, although prices are likely to remain stable.`
