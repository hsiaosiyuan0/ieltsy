# 第 4 章：非谓语动词 Non-finite Verbs

## §6.1 不定式 Infinitive

72. 不定式作主语 / 宾语 / 表语 / 定语 / 状语 / 补语 ★★★
73. 不带 `to` 的不定式 ★★（make / let / have / 感官动词 / had better 等）
74. 完成式不定式 ★★ — `to have done`
75. 进行式不定式 ★ — `to be doing`
76. 被动式不定式 ★★ — `to be done` / `to have been done`
77. 不定式的逻辑主语 ★★ — `for sb to do`
78. 形容词 + 不定式 ★★ — `eager / reluctant / likely to...`
79. 疑问词 + 不定式 ★★ — `how to...` / `what to...`
80. 不定式表目的 / 结果 / 原因 ★★★

## §6.2 动名词 Gerund

81. 动名词作主语 / 宾语 / 表语 / 定语 ★★★
82. 动名词的逻辑主语 ★ — `his coming` / `him coming`
83. 动名词 vs 不定式 ★★★ — remember / forget / regret / stop / try / mean / go on doing vs to do
84. 完成式动名词 ★ — `having done`
85. 被动式动名词 ★ — `being done` / `having been done`
86. 介词后必须用动名词 ★★★

## §6.3 分词 Participle

87. 现在分词作定语 / 状语 / 补语 ★★★
88. 过去分词作定语 / 状语 / 补语 ★★★
89. 完成式分词 ★★ — `having done` / `having been done`
90. 独立主格结构 ★★ — `Weather permitting, ...` / `All things considered, ...`
91. `with` 复合结构 ★★ — `with sth done / doing / to do`
92. 分词作话题主位（写作开头）★★ — `Compared with...` / `Given...` / `Considering...`
93. 悬垂分词（dangling modifier）的避免 ★★
