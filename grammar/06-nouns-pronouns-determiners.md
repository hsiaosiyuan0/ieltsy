## §11 名词与冠词 Nouns & Articles

132. 可数 vs 不可数 ★★★（**中文母语者大坑**：information / advice / equipment / furniture / research / knowledge / progress 不可数）
133. 集合名词 ★★ — family / team / committee / government / staff（单复数）
134. 不规则复数 ★★ — phenomena / criteria / data / media / analyses / hypotheses
135. 物质名词的可数化 ★ — `a coffee` / `two beers`
136. 抽象名词的具体化 ★ — `a beauty` / `a success`
137. `a / an` 元音音素 ★★★（an hour / a university）
138. `the` 用法 ★★★ — 特指 / 类指 / 唯一性 / 序数词 / 最高级 / 江河海洋山脉 / 国名复数
139. 零冠词 ★★★ — 类指复数 / 抽象名词 / 专有名词 / 三餐 / 月份 / 球类

## §12 代词 Pronouns

142. 反身代词 ★★ — 强调 / 反身用法

## §24 限定词与量词 Determiners & Quantifiers

### 限定词分类
265. 限定词总览 ★★ — 冠词 / 指示词 / 物主词 / 数词 / 不定限定词
266. 限定词组合限制 ★★ — 不能同时使用两个核心限定词（the my book ✗）→ 用 `of` 短语（a book of mine）
267. 数词位置 ★ — `all the three students` / `the last three students`

### 量词使用
268. `many` vs `much` ★★★ — 可数 vs 不可数
269. `few` vs `little` vs `a few` vs `a little` ★★★ — 无 a 表"几乎没有"，有 a 表"一些"
270. `a number of`（复数动词）vs `the number of`（单数动词）★★★
271. `a great deal of` / `a large amount of` ★★ — 不可数
272. `a great many` / `a large number of` ★★ — 可数复数
273. `plenty of` / `lots of` / `a lot of` ★★ — 可数 / 不可数皆可
274. `both` / `either` / `neither` ★★ — 两者
275. `all` / `every` / `each` ★★ — 三者及以上 / `every + 单数` / `all + 复数` / `each + 单数`
276. `enough` 位置 ★★ — `enough money` / `big enough`（修饰名词前置，修饰形/副后置）
