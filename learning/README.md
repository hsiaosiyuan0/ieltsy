# 学习日志

> 每日学习产物的归档目录。Db 是结构化数据源（进度、SM-2 状态），这里是**人类可读**的学习痕迹。

## 目录组织

```
learning/
├── days/                   每日学习产物（按 YYYY-MM-DD 一个文件夹）
│   └── YYYY-MM-DD/
│       ├── article.md      当日 AI 生成的短文 + 目标词覆盖 + 语法示例
│       └── session.md      默写 Q&A 全程记录 + 评分
└── mistakes/               错题本（跨日期累积，按时间倒序）
    ├── words.md            单词错题
    └── grammar.md          语法错题
```

## 使用方式

- 回顾某天学了什么：直接打开 `days/YYYY-MM-DD/article.md`
- 复习错题：打开 `mistakes/words.md` 看最近答错的词
- 让 Claude 出错题专项练习：在 Claude Code 里说"专项复习错题"
